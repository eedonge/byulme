Tooltips on mobile devices

The tooltip is a common graphical user interface element used in conjunction with a cursor. The user hovers the cursor over an item, without clicking it, and a tooltip may appear.
Tooltips do not appear on mobile operating systems, since there is no cursor.

Wikipedia
http://en.wikipedia.org/wiki/Tooltip

In user interface design, a modal window is a child window that requires users to interact with it before they can return to operating the parent application, thus preventing the workflow on the application main window.

Notification center

Notification Center is a feature in iOS and OS X that provides an overview of alerts from applications. It displays notifications until the user completes an associated action, rather than requiring instant resolution.
